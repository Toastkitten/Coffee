# 咖啡萃取與風味調整工具 (Web App)

React + Tailwind CSS 單頁應用，提供義式濃縮與手沖咖啡的參數計算與風味修正建議。

## 功能

- **義式濃縮**：粉重、粉液比、吸水率；計算目標萃取量與機器出水量；美式/拿鐵配方延伸。
- **手沖咖啡**：粉重、粉水比；目標總注水量；風味問題（太苦、太酸、想更甜厚、想更乾淨）的微調建議。

## 執行方式

```bash
cd web
npm install
npm run dev
```

瀏覽器開啟 http://localhost:5173。

## 建置

```bash
npm run build
npm run preview
```
